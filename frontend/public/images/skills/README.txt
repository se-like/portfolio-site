# スキル関連画像の配置について

このディレクトリには、スキルやテクノロジーに関連するアイコンや画像を配置してください。
以下の命名規則に従って画像を配置することをお勧めします：

1. スキルカテゴリーのアイコン：
   - frontend.svg - フロントエンド開発カテゴリー用アイコン
   - backend.svg - バックエンド開発カテゴリー用アイコン
   - devops.svg - インフラ・DevOpsカテゴリー用アイコン
   - database.svg - データベースカテゴリー用アイコン
   - other.svg - その他のスキルカテゴリー用アイコン

2. 個別のスキルやテクノロジーのアイコン：
   - react.svg - Reactのアイコン
   - nextjs.svg - Next.jsのアイコン
   - typescript.svg - TypeScriptのアイコン
   - rails.svg - Ruby on Railsのアイコン
   - nodejs.svg - Node.jsのアイコン
   - python.svg - Pythonのアイコン
   - aws.svg - AWSのアイコン
   - docker.svg - Dockerのアイコン
   - kubernetes.svg - Kubernetesのアイコン
   - postgresql.svg - PostgreSQLのアイコン
   - mongodb.svg - MongoDBのアイコン
   など

3. スキルページの背景画像：
   - skills-hero.jpg - スキルページのヒーローセクション用背景画像（推奨サイズ: 1920x600px）
   - skills-bg.jpg - スキルページの背景パターン画像

SVG形式のアイコンを使用すると、高解像度ディスプレイでも鮮明に表示されます。
各技術の公式ロゴを使用する場合は、ライセンスに注意してください。

画像を配置したら、開発サーバーを再起動する必要はありません。自動的に画像が読み込まれます。
