# プロフィール画像の配置について

このディレクトリには、以下のプロフィール画像を配置してください：

1. profile.jpg - メインのプロフィール画像（推奨サイズ: 500x500px）
   - ヘッダーセクションとAboutページで使用されます
   - プロフェッショナルな印象を与える画像を推奨します

2. profile-hero.jpg - ヒーローセクション用の大きなプロフィール画像（推奨サイズ: 1200x800px）
   - トップページのヒーローセクションで使用されます
   - 作業風景や、より広い背景を含む画像を推奨します

画像を配置したら、開発サーバーを再起動する必要はありません。自動的に画像が読み込まれます。
