# プロジェクト画像の配置について

このディレクトリには、各プロジェクトのスクリーンショットや関連画像を配置してください。
以下の命名規則に従って画像を配置することをお勧めします：

1. project-1.jpg, project-2.jpg, ... - プロジェクト一覧ページのメイン画像（推奨サイズ: 800x450px）
   - プロジェクト一覧ページで使用されます
   - プロジェクトの全体像が分かるスクリーンショットを推奨します

2. project-1-detail.jpg, project-2-detail.jpg, ... - プロジェクト詳細ページ用の大きな画像（推奨サイズ: 1200x675px）
   - プロジェクト詳細ページで使用されます
   - より高解像度で詳細が分かる画像を推奨します

3. project-1-additional-1.jpg, project-1-additional-2.jpg, ... - プロジェクトの追加画像
   - プロジェクト詳細ページのギャラリーセクションで使用されます
   - 異なる角度や機能を示す画像を推奨します

各プロジェクトの実際の名前に合わせて、より具体的なファイル名（例：ec-site.jpg, iot-platform.jpg）を使用することもできます。

画像を配置したら、開発サーバーを再起動する必要はありません。自動的に画像が読み込まれます。
